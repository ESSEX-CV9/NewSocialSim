## 视觉体系

- **设计令牌对齐 X**：纯黑底 #000、卡片 #16181c、边框 #2f3336、次级文字 #71767b，三强调色严格分工（蓝 #1d9bf0 操作 / 绿 #00ba7c 转发 / 粉 #f91880 赞）；Inter 字体、15px 正文、4px 基数尺寸、0.15/0.2s 统一过渡。令牌以 Tailwind 4 `@theme` 定义（`bg-x-*` 等工具类）。
- **图标库两次迁移**：先从 emoji 全量替换为 FontAwesome（npm 本地引入）；随后因 FA 免费版缺少房子/放大镜/地球的空心变体、无法实现导航"未选中空心/选中实心"双态，整体迁移到 Remix Icon（同为 npm 离线方案，全部图标具备 -line/-fill 双态）。
- **主题系统抽象**（commit `b585ac4`）：颜色改为两层结构——`[data-theme]` CSS 变量块定义主题，`@theme inline` 把 `x-*` 令牌映射到变量，全部工具类自动跟随主题。内置深色/浅色（浅色按 X 原版色值），ThemeContext 持久化到 localStorage。新增主题只需在 index.css 加一个变量块并在 theme/themes.ts 注册。
- 通用元素：毛玻璃 sticky 顶栏（rgba 背景 + blur(12px)，背景色随主题）、Tab 底部 56×4px 蓝色指示条、旋转加载圈、48px 图标空状态、互动按钮 hover 同色 10% 圆形气泡 + 整卡 3% 悬停叠加。

## 页面与交互改动

### 个人资料页（X 布局）
53px 标题栏（返回 + 名字 + 帖子数）→ 200px 纯色 Banner（图片上传留待媒体系统）→ 80px 头像以 4px 主题背景色描边骑在交界处 → 简介区（加入时间按世界模拟时间显示年月）→ 帖子/回复/媒体/喜欢四 Tab（懒加载；媒体为空态占位）。帖子 Tab 后改为"原创 + 本人转发"的时间线。

### 首页双流
"为你推荐 / 关注"双 Tab；点击已激活的 Tab 弹出下拉：为你推荐可切"全站"，关注可切"按最新 / 按热度"。关注流剔除本人内容；用户没有关注任何人时显示推荐关注列表（可直接关注）。

### 通知页
- "全部 / 提及"双 Tab（提及 = @mention + 回复）。
- 卡片 X 样式：左侧 26px 类型图标，右侧头像行 + 文案行 + 帖子内容预览；文字三层层级（动作文案主色最实、摘要 13px 60% 主色居中、时间次级色最浅）。
- 客户端聚合相邻同类通知：同人多帖（"点赞了你的 N 个帖子"）、同帖多人（"A 和另外 N 人赞了你的帖子"）、连续关注；回复/引用/提及不聚合。
- 头像堆叠（facepile）：组内 actor 按"被我关注 > 粉丝多 > 最新"排序，领头完整置顶层，后续各遮一半（人数越多叠得越紧：2 人露 16px / 3 人露 12px / 4 人露 8px），第 3、4 个渐变淡出，上限 4 个。
- 修复：点击通知跳转后未标记已读，原先只能靠"全部已读"清除高亮；现点击时批量标记组内未读项。

### 右边栏重构

- 新增公开接口 `GET /api/search/trends?limit=`（挂 search 模块）：统计近 7 个模拟日内帖子正文的 #话题 排行，返回 `{ trends: [{ tag, postCount }] }`。时间窗口用世界时钟（`clock.now()`）界定；SQL 以 `LIKE '%#%'` 预筛后在 service 层按与前端一致的正则 `/#[^\s#@]+/g` 解析；同帖同话题只计一次（验证：同帖重复话题计 1，软删除后从趋势消失）；按小写聚合、保留首现大小写。
- 新组件 RightSidebar 取代 Layout 内联右栏，自上而下：搜索框（列内吸顶）→ "有什么新鲜事"趋势卡（60 秒轮询，点击话题跳搜索页）→ "推荐关注"卡（复用 suggested-users 查询缓存，可直接关注）→ "当前世界"折叠卡 → 装饰性页脚（服务条款/隐私政策/Cookie 政策/©，无动作）。
- 世界卡默认折叠，折叠态只显示世界名与跳动模拟时钟，点击展开描述与流速；展开状态仅组件内存，刷新即回折叠。该改动源于用户反馈：常驻展开的世界信息"容易出戏"。
- 右栏 `overflow-y-auto` 可滚动并以 `.no-scrollbar` 工具类隐藏滚动条；首页空关注流的内联推荐关注组件保留未动。
- i18n 新增 `trends.*`、`footer.*` 双语文案。
